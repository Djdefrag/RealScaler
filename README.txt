Hi, thanks for downloading RealScaler :D.

PREREQUISITES.
1. (If not already installed) install Visual-C-Runtimes: 
   - https://www.techpowerup.com/download/visual-c-redistributable-runtime-package-all-in-one/

2. (If not already installed) install Directx:
   - https://www.microsoft.com/it-it/download/details.aspx?id=35 

INSTALLATION.
1. Delete previous directory of RealScaler if you have already installed
2. Extract RealScaler from the .zip

HOW TO USE.
1. Open RealScaler directory
2. Execute RealScaler.exe
